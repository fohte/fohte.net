久々に買い換えると気持ちがリフレッシュできて良い。新鮮。

主な買い換え理由はバッテリー劣化とストレージ容量を増やしたかったから。
そろそろ買い換え時だったし、15 ではついに USB-C になってケーブル類を統一できて嬉しそうだったので、このタイミングで買い換えた。

# 購入編

今まで iPhone を予約してまで買ったことがなかったのだが、今回は半年前くらいから買い換えることを決心していたので、Apple Store (Web) で予約購入した。

事前予約までして準備万端だったのだが、予約開始時刻なのにそれを忘れて悠長に風呂に入っており、完全に出遅れた。

# 使用感

ということで、先日 10/6 に無事届いたので、2 日使った所感を書いておく。iPhone 11 Pro との比較して書いている。

## 嬉しいポイント

- 120 fps (ProMotion) がかなり良い
  - ゲームするわけでもないけど、基本的な操作がヌルサクになってかなり体験が良い
    - CPU 性能の向上と相まって体験が良くなっている
  - 特に、高速にスクロールしても文字を読めるのが良い
- Dynamic Island が面白い
  - 音楽再生中にアルバムアートが表示されたりして楽しい
- Face ID が認識されやすく、かつ速くなった
  - 画面を注視していなくても Face ID が通るようになった
  - Face ID の処理開始から認識されるまでの時間が体感でかなり短くなった
  - Face ID はスリープからのロック解除やパスワードマネージャーのロック解除で多用するので、そこが改善されたのはとても良い
- スピーカーの低音の質が良くなった
- MagSafe が便利
  - MagSafe スタンドを買ったら iPhone の置き場所に困らなくなったし、充電も勝手にされていて便利
  - もう充電ケーブルを使わなくなった
- (15 Pro と関係ないけど) ストレージ容量が 64 GB -> 256 GB に増えた
  - 原神がインストールできるようになった
    - 基本的に PC でやるけど、樹脂が溢れそうなときに出先で濃縮できるのが便利 (些末すぎる)

## まだ使いこなせていないポイント

- USB-C は有効活用できてない
  - 結局 MagSafe で充電できてしまうので、ケーブルで充電していない
    - データ転送時は嬉しいのかも? (ただ iPhone でケーブル繋いでデータ転送をしたことはない)
  - 身の回りのケーブルがほとんど USB-C ではないので、ケーブルの買い換えが必要になってしまった
  - すぐに充電したいときに USB PD で急速充電できるのは便利だった
- 5G が使えるようになった
  - が、5G の恩恵は特に感じていない… 表示が変わってなんとなく速そうで嬉しい、くらいの気持ち
- スタンバイモード
  - MagSafe スタンドと合わせてスタンバイモードが使えるようになった
  - が、時刻をでかでかと表示させているだけであまり嬉しさがない
    - 時計は別にあるし…
- アクションボタン
  - 常にサイレントで使っているからサイレント切り替えを割り当てても嬉しくない
  - カメラもコントロールセンターやロック画面から起動できるし…
  - という感じで有効活用できそうなのにうまい活用方法が思いつかず使いこなせていない

## 全体的な感想

iPhone 8 -> 11 Pro に乗り換えたときほどの感動はなかったけど、地味ながらも便利なことが増えているなという印象。

個人的に嬉しいのは ProMotion と Face ID の性能向上だった。Touch ID の復活を待望していたけど、Face ID がこれだけ速ければ Touch ID なくてもいいなと思えた。

あと、カメラについては触れていないけど、まだカメラ使うタイミングが無くてまだなんとも言えない。カメラがデカいくらいの感想しかない。

# 最後に

iPhone 15 Pro の予約争奪戦に負けた話と、使い始めて 2 日目の所感を書いた。

バッテリー劣化とストレージ容量不足に悩んでいなければ買っていなかったかも、とは思いつつ、地味ながらも便利なところは増えていて、買い換えたことには後悔はしていない。円安で高かったけど…。
