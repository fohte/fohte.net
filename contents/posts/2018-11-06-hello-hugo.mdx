[fohte.hateblo.jp](//fohte.hateblo.jp) から転生した。

Yahoo! ブログ → FC2 ブログ → ライブドアブログ → はてなブログ → Hugo (イマココ) という感じで変遷している。

今回はブログサービスを用いず、[Hugo](https://gohugo.io/) という静的サイトジェネレーターでブログを作成し、Netlify でホスティングしている。

なぜ Hugo に乗り換えたかというと、

- 持て余していた [fohte.net](//fohte.net) ドメインを活用したかった
- Vim でブログを書きたかった
- GitHub Flavored Markdown でブログを書きたかった
- `git push` すればブログを公開できるようにしたかった

という理由がある。

ドメインの活用についてははてなブログでも可能だったが、Hugo はブログだけに留まらず、単純なポートフォリオサイトも作成できるのが利点。

そういうわけで、今後はブログはこちらで書いていくのでよろしくお願いします。
