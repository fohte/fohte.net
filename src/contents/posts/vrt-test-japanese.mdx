# 日本語コンテンツのテスト

この投稿は、日本語フォントとレイアウトが正しく表示されることを確認するためのものです。

## なぜ重要か

日本語には以下の特徴があります：

- 漢字、ひらがな、カタカナの3つの文字体系
- 英数字との混在
- 異なる文字幅（全角・半角）

## サンプルテキスト

吾輩は猫である。名前はまだ無い。どこで生れたかとんと見当がつかぬ。何でも薄暗いじめじめした所でニャーニャー泣いていた事だけは記憶している。

### 技術用語の混在

ReactやNext.jsを使って、日本語のWebアプリケーションを開発する際の注意点。

```typescript
// 日本語コメントのテスト
const greeting = (name: string): string => {
  return `こんにちは、${name}さん！`
```
